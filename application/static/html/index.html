QuikImpakt


What is QuikImpakt?

QuickImpakt is a web tool to extract abstracts from PubMed's vast collection of scientific documents and create word meshes. It is helpful for presentations to summarize data.

How does it work?

QuickImpakt is created with Python and is hosted with the Flask framework. When you type any scientific topic into the search bar and press the Build button, the following happens:

• The topic is searched up on PubMed's website.
• The first 7 articles are requested.
• Each of their articles' abstracts are taken, written to a text file, and joined into one paragraph.
• The top 30 most common words are taken from the paragraph.
• Those top 30 words are put into a solid black picture with a random color (that isn't dark). Common words like "and" or "it" are not included.
